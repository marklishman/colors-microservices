# Green

## Features

* Spring Data JPA
* Spring HATEOAS

---

# Getting Started

### Postgres
* `docker start ref-db`
* Create a schema called `green`
* Run the `scripts/sql/schema.sql` script in this schema.
* Run the `scripts/sql/data.sql` script in this schema.

### HTTP
* Run the HTTP scripts in the `/scripts/http` directory
